Tested with Unity 2022.3 on Windows 10/11
- Open in Safe Mode (to install dependencies before compiling)
- Install the dependencies
- Build Menge and Plugins:
    git clone https://github.com/MichelangeloDiamanti/Menge.git
    git pull origin master 
    follow build instructions on the repo's README
